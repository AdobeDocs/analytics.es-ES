# Tipos de explorador

Se refiere al tipo de explorador que utiliza el visitante (por ejemplo, Internet Explorer, Mozilla Firefox, Google Chrome, etc.).

Los tipos de exploradores no identificados aparecen como "no especificado".

Por ejemplo:

| Google |
|---|
| Mozilla |
| Microsoft |
| Apple |
| Opera |
| No especificado |
| BlackBerry |
| UC Web |
| Otro |
| Nokia |
