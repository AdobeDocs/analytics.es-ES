# Importación del explorador y FTP

Una vez copiados los datos adecuados en el archivo de plantilla de clasificaciones, puede procederse a la importación (carga) en los servidores de recopilación de datos de Adobe.

>[!NOTE] No es necesario descargar (con la función de exportación) los datos de clasificación existentes antes de cargar los nuevos datos. Solo es necesario cargar un nuevo archivo que contenga las nuevas filas de datos. Este método mejora el rendimiento.

See [General File Structure](/help/components/c-classifications2/c-classifications-importer/c-saint-data-files.md) for important information about the structure and limitations of uploaded files.

Consulte [Problemas comunes de carga de clasificaciones](https://helpx.adobe.com/es/analytics/kb/common-saint-upload-issues.html) para obtener ayuda sobre la resolución de problemas.
