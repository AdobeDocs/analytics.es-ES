# Año

La dimensión &#39;Año&#39; informa el año en que se produjo una métrica determinada. El primer valor de dimensión es el primer año del intervalo de fechas y el último valor de dimensión es el año más reciente del intervalo de fechas.

## Rellenar esta dimensión con datos

Esta dimensión funciona de forma predeterminada para todas las implementaciones. Si un grupo de informes contiene datos, esta dimensión funciona.

## Valores de dimensión

Los valores de dimensión incluyen un año determinado, con las fechas incluidas entre paréntesis.
