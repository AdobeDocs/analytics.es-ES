# Mes

La dimensión &#39;Mes&#39; informa el mes en que se produjo una métrica determinada. El primer valor de dimensión es el primer mes del intervalo de fechas y el último valor de dimensión es el último mes del intervalo de fechas. Esta dimensión es vital para los informes de tendencias, ya que le permite ver las métricas a lo largo del tiempo.

## Rellenar esta dimensión con datos

Esta dimensión funciona de forma predeterminada para todas las implementaciones. Si un grupo de informes contiene datos, esta dimensión funciona.

## Valores de dimensión

Los valores de dimensión incluyen el mes y el año de una fecha determinada.
