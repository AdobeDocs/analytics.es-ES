# Buscar métricas

Las métricas se pueden localizar en Analysis Workspace del mismo modo que se localizan otros tipos de componentes.

Para obtener información sobre cómo localizar rápidamente componentes, consulte [Buscar, filtrar y ordenar la lista de componentes](https://experienceleague.adobe.com/docs/analytics/analyze/analysis-workspace/components/analysis-workspace-components.html#search%2C-filter%2C-and-sort-the-component-list) en [Información general sobre componentes](/help/analyze/analysis-workspace/components/analysis-workspace-components.md).
