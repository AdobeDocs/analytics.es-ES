# Uso del modo FTP pasivo

La diferencia entre el modo activo y pasivo del FTP controla el modo en que se establecen las conexiones de puerto. Esta elección tiene efectos en el cortafuegos.

Adobe utiliza el FTP pasivo, que es una forma más segura de transferencia de datos, en la que el flujo de datos se configura e inicia mediante el cliente del Programa de transferencia de archivos (FTP) en lugar de hacerlo mediante el programa de servidor FTP. Si tiene dificultades para conectarse al FTP de Adobe, asegúrese de utilizar el modo pasivo.
