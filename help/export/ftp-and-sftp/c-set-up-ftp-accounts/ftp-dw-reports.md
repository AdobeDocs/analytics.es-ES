# Informes entregados por Data Warehouse

>[!NOTE]
>
>La siguiente información está relacionada con los tipos de destino FTP y SFTP. FTP y SFTP son tipos de destino heredados. Al configurar un informe de Data Warehouse, debe utilizar un tipo de destino de nube, que es más seguro. Para obtener más información sobre la configuración de tipos de destinos de nube para un informe de Data Warehouse, consulte [Crear una solicitud de Data Warehouse](/help/export/data-warehouse/create-request/t-dw-create-request.md).

Data Warehouse le permite solicitar informes para mostrar las relaciones de datos avanzadas a partir de datos sin procesar basándose en sus preguntas únicas.

La entrega por FTP, en este caso, está disponible como parte de la configuración de [!UICONTROL Opciones de envío avanzadas]. Si utiliza un FTP, necesitará una cuenta interna de FTP ya existente o una cuenta de FTP de Adobe. Adobe no configura automáticamente la cuenta de FTP a través de la interfaz de Data Warehouse. La entrega por FTP debe ser inmediata, en función del tamaño del archivo y del número de informes que se procesen.

Para obtener información sobre los límites de FTP y la retención de datos, consulte [Límites de FTP y retención de datos](/help/export/ftp-and-sftp/ftp-limits.md).
