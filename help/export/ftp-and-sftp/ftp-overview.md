# Utilizar FTP y SFTP con Adobe CX Enterprise

Aprenda a usar los protocolos FTP y SFTP para transferir archivos entre equipos y servidores.

El FTP se puede usar con las soluciones y los productos de Adobe para:

* Cargar grandes conjuntos de datos de clasificación a Analytics.
* Cargar atributos del cliente a CX Enterprise para usarlos en soluciones.
* Utilice Analytics para crear y gestionar orígenes de datos basados en FTP, que aprovecha la transferencia de archivos FTP para importar datos históricos y sin conexión a CX Enterprise

En estos artículos no se detallan los pasos para usar FTP con cada producto. Aquí recomendamos las prácticas recomendadas para usar el FTP con Adobe.
