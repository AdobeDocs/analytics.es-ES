# Clasificaciones

{{legacy-arb}}

Los desgloses de clasificación se utilizan para asignar datos de informes de análisis a propiedades relacionadas. Las clasificaciones se pueden utilizar para varios fines, pero se utilizan principalmente para clasificar códigos de seguimiento de campaña, tanto internos como externos, e ID de productos.

Por ejemplo, un informe de [!UICONTROL Campaign] puede tener una clasificación de [!UICONTROL Manager] y una clasificación de [!UICONTROL Outlet]. Estas clasificaciones se pueden ordenar siguiendo una jerarquía determinada, y cada grupo de valores del informe se puede desglosar en subgrupos.

Puede acceder a las clasificaciones de la misma manera que accede a los demás informes de un grupo de informes, y las dimensiones se muestran como un informe de nivel superior dentro del grupo. Si un informe tiene clasificaciones, se crea un subgrupo de menús para recopilar el informe y todas sus clasificaciones en la misma ubicación. El informe se muestra como la última entrada de la lista entre las clasificaciones.

Consulte [Clasificaciones](/help/components/classifications/classifications-overview.md).
