# Mensajes de diagnóstico

{{legacy-arb}}

Los mensajes de diagnóstico ofrecen información sobre el estado de la solicitud cuando se actualiza.

![Icono de marca de verificación verde que indica que la solicitud se actualizó correctamente.](assets/icon_notice_success.gif) Indica que la solicitud se ha actualizado correctamente desde que se abrió este libro. La columna de diagnóstico proporciona otra información sobre la solicitud, incluido el tiempo utilizado por la solicitud, el tiempo para asignar la solicitud y si la solicitud utiliza información almacenada en caché anteriormente.

![Icono de triángulo rojo con signo de exclamación que indica que la actualización de la solicitud ha fallado.](assets/icon_notice_warn.gif) Esta alerta se muestra cuando una solicitud no se actualiza correctamente. El error se anota en la columna de diagnóstico que señala la causa probable del error. Consulte la [sección de solución de problemas](/help/analyze/legacy-report-builder/troubleshoot.md) para obtener ayuda con los mensajes de error seleccionados.
