# Resumen de solicitudes de copia

{{legacy-arb}}

Es posible pegar las celdas asignadas con varias solicitudes y pegar el contenido en una zona seleccionada y vacía de la hoja de cálculo.

Una vez copiadas las celdas, Report Builder y Excel determinan el área necesaria para pegar la cantidad mínima de copias. Si el área es lo suficientemente grande, al pegar se crea una copia de todas las solicitudes, donde cada solicitud pegada tiene la misma disposición espacial y el mismo formato que en las solicitudes originales.

Esto se conoce como propagación de la solicitud. Es una forma fácil y rápida de crear un informe largo. En primer lugar, Report Builder propaga las solicitudes pegándolas todas en las celdas de la zona de pegado de destino y, a continuación, actualiza las celdas en función de las fechas del informe establecidas para las solicitudes.
