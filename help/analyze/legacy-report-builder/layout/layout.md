# Diseño: Paso 2 del Asistente para solicitudes

{{legacy-arb}}

La ventana [!UICONTROL Asistente para solicitudes: Paso 2] permite configurar el diseño del informe, el formato del encabezado y el filtrado de dimensión para la salida. Se puede utilizar un diseño de tabla dinámica para configurar la solicitud, de modo similar a como se crea una tabla dinámica de Excel.

Como alternativa, es posible seleccionar manualmente las celdas en las que desea mostrar los datos del informe utilizando el [Diseño personalizado](/help/analyze/legacy-report-builder/layout/configure-the-custom-layout.md).
