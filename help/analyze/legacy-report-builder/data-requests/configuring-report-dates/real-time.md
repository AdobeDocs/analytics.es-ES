# Informes en tiempo real

{{legacy-arb}}

Muestra el tráfico de la página web y clasifica las vistas de la página en tiempo real, de modo que puede comprender más rápidamente las tendencias de su sitio.

Para los informes en tiempo real, Analytics utiliza métricas de alta frecuencia y análisis de sitio para informar de forma visual sobre el tráfico y las tendencias de vistas de páginas de noticias dinámicas y sitios web comerciales. El informe en tiempo real comprende las tendencias minuto a minuto, pocos segundos después de haber recopilado los datos. Recopila y transmite datos a la interfaz mediante correlación en tiempo real, seguimiento de contenido y conversión.

Puede:

* Cree hasta tres informes en tiempo real por grupo de informes, utilizando las métricas, dimensiones y clasificaciones existentes. Utilice las dimensiones secundarias para establecer una correlación (o desglose) respecto a la primaria.
* Agregue tres dimensiones (o clasificaciones) por informe (uno principal y dos secundarios), además de las métricas de todo el sitio.
* Utilice cualquier evento personalizado, evento del carro de compras o instancia.
* Permite ver hasta 20 horas de datos históricos en tiempo real.

Real-time respeta los permisos de usuarios y grupos. Por ejemplo, si no tiene derechos para ver los ingresos, no puede ver un informe en tiempo real que incluya datos de ingresos. No se admiten eVars (métricas de conversión). Los permisos en tiempo real están habilitados en [Herramientas de administración](/help/admin/tools/manage-rs/edit-settings/realtime/t-realtime-admin.md).

>[!NOTE]
>
>[!UICONTROL Informes en tiempo real] no son lo mismo que la función [Incluir datos actuales](/help/analyze/legacy-report-builder/options.md) (Actualización de los datos) lanzada anteriormente, que redujo la latencia de los informes estándar al mostrar los datos antes de que finalizara.
