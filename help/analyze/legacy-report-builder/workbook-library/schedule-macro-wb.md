# Programar libros con las macros habilitadas

{{legacy-arb}}

Report Builder de Adobe admite el formato .xlsm que le permite programar libros con las macros habilitadas.

Esto resulta útil si necesita programar, procesar y recibir libros con las macros habilitadas de forma segura.

>[!IMPORTANT]
>
>Aunque Report Builder le permita programar libros con las macros habilitadas, no se pueden accionar durante cada una de las ejecuciones programadas. Solo se ejercen cuando el libro está abierto en la aplicación nativa de Microsoft Excel.

Los libros programados con macros sólo se pueden entregar en formato habilitado para macros (.xlsm), ya que todos los demás formatos admitidos (xls, xlsx, pdf, word, csv o txt) quitarían las macros del libro.
