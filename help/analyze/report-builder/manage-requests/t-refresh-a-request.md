# Actualizar solicitudes

Las solicitudes se actualizan una vez editadas o filtradas, o si se desean ver los datos más recientes. Se pueden seleccionar varias solicitudes de una hoja de cálculo y actualizarlas al mismo tiempo.

Si aumenta el intervalo de fechas de una solicitud y, a continuación, esta se actualiza, Report Builder añadirá celdas para albergar el número añadido de períodos.

1. Cree y ejecute una solicitud.
1. After you have changed the request, right-click the request, then choose **[!UICONTROL Refresh Request]** from the shortcut menu. (Alternatively, click in the request, then click **[!UICONTROL Refresh]** in the toolbar.)

   El sistema muestra el [!UICONTROL Refresh Requests] formulario:

   **Fecha ahora:** fecha en la que se desea basar la actualización. También se conoce como fecha [!UICONTROL As Of] .

   **Establecer en fecha actual en futuras acciones de actualización:** Al habilitar esta opción, se indica al creador de informes que utilice siempre la fecha actual como fecha [!UICONTROL Now] (o [!UICONTROL As Of]). You can revert this setting by clicking [!UICONTROL Options] in the toolbar.
1. Haga clic en **[!UICONTROL OK]**.

   When a refresh is successful, the system reports this with a green icon on the [!UICONTROL Request Manager].
