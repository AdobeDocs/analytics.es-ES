# Implementación del Activity Map

El Activity Map se puede implementar con una casilla de verificación en la extensión de Adobe Experience Platform Launch para Adobe Analytics, en [Seguimiento de vínculos](https://experienceleague.adobe.com/docs/launch/using/extensions-ref/adobe-extension/analytics-extension/overview.html?lang=en#link-tracking).
