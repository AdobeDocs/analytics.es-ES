# Incorporación de Adobe Analytics

Bienvenido a la serie de incorporación a la experiencia digital de Adobe Analytics.

Adobe Analytics permite a las organizaciones recopilar datos y obtener información procesable de cualquier interacción digital del cliente. Con un análisis profundo, informes versátiles e inteligencia predictiva, las organizaciones obtienen la información que necesitan para crear mejores experiencias de cliente.

El Recorrido de Adobe Analytics comienza con la incorporación de las configuraciones y el acceso de los usuarios adecuados. Los recursos siguientes optimizan esa incorporación y facilitan el éxito de usted y su organización con Adobe Analytics.

Vea los siguientes vídeos para completar el recorrido de incorporación de Adobe Admin Console para usuarios y acceder a la configuración de Adobe Experience League para obtener documentación y asistencia, y finalmente, el propio Adobe Analytics.

## Adobe Admin Console

>[!VIDEO](https://video.tv.adobe.com/v/3476921/?learn=on&enablevpops)

Incorpórese a [Adobe Admin Console](https://adminconsole.adobe.com/es) para administrar las asignaciones de productos para su organización. Agregar, quitar o modificar el acceso de usuario.

## Administración de usuarios de Adobe Admin Console

>[!VIDEO](https://video.tv.adobe.com/v/3476898/?learn=on&enablevpops)

Incorpórese a [Administración de usuarios de Adobe Admin Console](https://adminconsole.adobe.com/redirect@AdobeOrg/users) para administrar usuarios, grupos y perfiles. Obtenga información sobre cómo asignar una función de producto a un usuario o grupo.

## Presentación de un caso de asistencia

>[!VIDEO](https://video.tv.adobe.com/v/3476787/?learn=on&enablevpops)

Incorpórese a Adobe Experience League para [plantear un caso de asistencia, revisar los casos activos y anteriores](https://experienceleague.adobe.com/home#support).
