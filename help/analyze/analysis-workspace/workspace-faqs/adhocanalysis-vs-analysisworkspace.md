# Analysis Workspace comparado con los Ad Hoc Analysis

Compara la terminología y tareas de Análisis ad hoc con el espacio de trabajo de Análisis.

Espacio de trabajo de Análisis incorpora gran parte de la funcionalidad de Análisis ad hoc en el flujo de trabajo del explorador. Aunque algunos términos y características siguen siendo los mismos entre los productos, hay algunos términos y enfoques nuevos de la análisis que se introducen en Espacio de trabajo de Análisis.

Para una comparación técnica de las características clave y los requisitos del sistema entre estos dos productos, vaya [aquí](https://marketing.adobe.com/resources/help/en_US/reference/analytics-product-comparison.html).

## Comparación de terminología clave  {#section_6109406B83B043A18E46D38F130B1D2E}

| Ad Hoc Analysis | Analysis Workspace |
|--- |--- |
| Proyecto | Espacio de trabajo o proyecto |
| Espacio de trabajo | Panel |
| Información general de la tienda de aplicaciones | Tabla de forma libre |
| Gráficos/Gráfico | Visualización |
| Jerarquía: Proyecto > Workspace > Informes | Jerarquía: Proyecto > Paneles > Tablas |
| Plantillas de informe de clasificación, tendencias y totales | Visualización de tabla improvisada |
| Plantilla de flujo | Visualización de flujo |
| Abandono | Visualización de visitas en el orden previsto |
