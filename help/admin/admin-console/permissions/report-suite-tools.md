# Herramientas de grupos de informes listados en los permisos de Perfil de producto

Utilice esta página para comprender cada elemento de permiso en Herramientas de grupos de informes.

| Elemento de permiso de las herramientas de grupos de informes | Descripción |
|------|------|
| Data Warehouse | No se utiliza. En su lugar, utilice el informe Almacén de datos personalizado. |
| SiteCatalyst | No se utiliza. En su lugar, utilice «Acceso a Analysis Workspace». |
| Grupos de informes (lectura) | No se usa. |
| Grupos de informes (escritura) | No se usa. |
| Administración del tráfico | No se usa. |
| Administración de conjuntos de informes | Otorga acceso al Administrador de grupos de informes, pero no permite ningún cambio. |
| Resumen de la cuenta | Otorga acceso a General &gt; Configuración general de cuenta en el Administrador de grupos de informes. |
| Filtros de dirección URL | Otorga acceso a General &gt; Filtros URL internos en el Administrador de grupos de informes. |
| Calendario personalizado | Otorga acceso a General &gt; Personalizar calendario en el Administrador del grupo de informes. |
| Búsqueda de pago | Otorga acceso a General &gt; Detección de búsqueda paga en el Administrador de grupos de informes. |
| Personalización de menús | Otorga acceso a General &gt; Personalizar menús en el Administrador del grupo de informes. |
| Configuración de informes en tiempo real | Otorga acceso a tiempo real en el Administrador del grupo de informes. Utilice el elemento de permiso'Informe realtime'para otorgar acceso al propio informe. |
| Configuración de vídeo | No se usa. |
| Variables de tráfico | Otorga acceso a Tráfico &gt; Variables de tráfico en el Administrador de grupos de informes. |
| Canales | Concede acceso a los canales de mercadotecnia, incluidos el Administrador de canales de mercadotecnia, las Reglas de procesamiento de canal de mercadotecnia y Caducidad de canal de mercadotecnia en el Administrador de grupos de informes. |
| Costos | Otorga acceso a Canales de mercadotecnia &gt; Costos de canal de mercadotecnia en el Administrador de grupos de informes. |
| Variables de conversión | Otorga acceso a Conversión &gt; Variables de conversión en el Administrador de grupos de informes. |
| Métodos de búsqueda | Otorga acceso a Conversión &gt; Métodos de búsqueda en el Administrador de grupos de informes. |
| Visitante único | Otorga acceso a Conversión &gt; Variable de visitante único. Normalmente no se utiliza en implementaciones modernas. |
| Eventos de éxito | Otorga acceso a Conversión &gt; Eventos de éxito en el Administrador de grupos de informes. |
| Variables de lista | Otorga acceso a Conversión &gt; Variables de lista en el Administrador de grupos de informes. |
| Medidas predeterminadas | Otorga acceso a la Configuración de grupos de informes individuales &gt; Métricas predeterminadas en el Administrador de grupos de informes. |
| Reglas de procesamiento | Otorga acceso a General &gt; Reglas de procesamiento y General &gt; Reglas Bot en el Administrador de grupos de informes. |
| Configuración de Advertising Analytics | No se usa. |
| Detección de anomalías | Otorga acceso a Detección de anomalías en Analysis Workspace. |
| Análisis de contribución | Otorga acceso al menú de clic derecho'Ejecutar análisis de contribución'en Analysis Workspace. |
| Informe de canal | Otorga acceso al informe Canales de mercadotecnia &gt; Información general de canal en Informes y análisis. |
| Informe en tiempo real | Otorga acceso a Métricas del sitio &gt; Tiempo real en Informes y análisis. Utilice el elemento de permiso'Configuración del informe en tiempo real'para otorgar acceso a la configuración de este informe. |
| Páginas de bots | Otorga acceso a Métricas del sitio &gt; Bots &gt; Informe Páginas de bots en Informes y análisis. |
| Bots | Otorga acceso a Métricas del sitio &gt; Bots &gt; Informe Bots en Informes y análisis. |
| Administrador de fuentes de datos | Otorga acceso a Administración &gt; Fuentes de datos en la navegación superior de Analytics. |
| Informe del Data Warehouse personalizado | Otorga acceso a Herramientas &gt; Almacén de datos en la navegación superior de Analytics. |
| Visitas de retorno diario | Otorga acceso a Retención de visitantes &gt; Visitas de retorno diario en Informes y análisis. |
| Administrador de fuentes de datos | Otorga acceso a Administración &gt; Fuentes de datos en la navegación superior de Analytics. Requiere el elemento de permiso'Mgmt de grupo de informes '. |
| Excluir por dirección IP | Otorga acceso a Administración &gt; Excluir por IP en la navegación superior de Analytics. |
| Últimos 100 visitantes | No se usa. |
| ClickMap heredado | No se utiliza. Utilice Mapa de actividades en Herramientas de Analytics en su lugar. |
| Instalación heredada de clickmap | No se utiliza. Utilice Mapa de actividades en Herramientas de Analytics en su lugar. |
| Visitas de retorno | Otorga acceso a Retención de visitantes &gt; Visitas de retorno en Informes y análisis. |
| Informe general de aplicaciones móviles | No se usa. |
| Clasificaciones | Otorga acceso a todas las opciones de clasificación en el Administrador de grupos de informes. This permission item includes: <br>Traffic &gt; Traffic Classifications<br>Conversion &gt; Conversion Classifications<br>Conversion &gt; Classification Hierarchies<br>Marketing Channels &gt; Marketing Channel Classifications<br>App Management &gt; App Classifications<br>Media Management &gt; Media Classifications<br>AEM &gt; AEM Assets Classifications<br>Adobe Campaign &gt; Adobe Classifications |
| Informes recomendados | Otorga acceso a Métricas del sitio &gt; Mis informes recomendados en Informes y análisis. |
| Informe breve de resumen de la empresa | No se usa. |
| Imagen | No se usa. |
| KPI/informe breve de medición | No se usa. |
| Totales de los grupos de informes | No se usa. |
| Informe breve de texto | No se usa. |
| Informe breve de resumen de uso | No se usa. |
| Recursos Web | No se usa. |
| Social | No se usa. |
