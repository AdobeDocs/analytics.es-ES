# Informes del Report Builder

Administre la licencia asignada a los usuarios Report Builder.

A cada licencia de Report Builder asignada a un usuario se le asignan automáticamente diez tareas programadas. Este número de tareas se puede ajustar y reasignar. El número de tareas programadas no puede superar el número de tareas para cada usuario. Las tareas programadas por encima de este número no se ejecutarán.

Puede conceder al Report Builder acceso a cualquier [!DNL Analytics] al administrar perfiles de producto en la variable [Adobe Admin Console](/help/admin/admin-console/home.md).
