# Métricas predeterminadas

[!UICONTROL **Editar configuración**] > [!UICONTROL **Configuración de grupo de informes individual**] > [!UICONTROL **Métricas predeterminadas**]

Analytics muestra un conjunto predeterminado de métricas en todos los informes y proyectos, a menos que algún usuario seleccione un conjunto personalizado de métricas. Las medidas seleccionadas se muestran a todos los usuarios del grupo de informes asociado. Solo se pueden actualizar las medidas predeterminadas de un grupo de informes cada vez. Esta configuración no garantiza el orden en el que se muestran las métricas predeterminadas.

Para obtener más información, consulte:

* [Métricas en Analysis Workspace](/help/analyze/analysis-workspace/components/apply-create-metrics.md)

* [Métricas calculadas &#x200B;](/help/components/calculated-metrics/cm-overview.md)
