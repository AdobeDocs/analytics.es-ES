# Notas de la versión de Adobe Analytics de 2026

Obtenga información sobre las últimas actualizaciones de la versión para los [productos de Adobe Experience Cloud](https://business.adobe.com/es/products/adobe-experience-cloud-products.html). Obtenga la documentación de autoayuda más reciente: tutoriales y cursos sobre Experience League.

## Febrero de 2026 {#feb26}

| Función y descripción | [Inicio del despliegue](releases.md) | [Disponibilidad general](releases.md) |
| ----------- | ------- | ---- |
| **Fuentes de datos mejoradas** <br/>Las siguientes mejoras ya están disponibles para las fuentes de datos:<ul><li>Experiencia del usuario mejorada.</li><li>Nueva experiencia para crear y administrar plantillas de columna.</li><li>Ahora puede elegir cuándo crear una plantilla de columna para reutilizarla en futuras fuentes de datos. También puede eliminar, editar y copiar plantillas.<br/>Anteriormente, cada fuente de datos creada tenía como resultado una nueva plantilla de columna, lo que generaba un gran número de plantillas de columna no utilizadas y ninguna forma de eliminarlas o administrarlas.</li><li>Añadir y filtrar por etiquetas.</li><li>Ver el historial de los trabajos de fuentes de datos. (Cuándo comenzó el período de solicitud, cuándo comenzó, terminó, etc.).</li><li>Reenviar o volver a procesar fuentes de datos. (Desde la página Historial de trabajos)</li><li>Permitir visitas tardías. Ahora puede incluir datos que llegaron después de que el trabajo de fuente de datos terminara de procesar los datos dentro de la frecuencia de creación de informes establecida.</li><li>Al elegir una fecha de finalización para una fuente de datos, la fecha de finalización que elija se incluye como el último día de la fuente de datos.<br/>Anteriormente, la fecha de finalización se excluía de la fuente de datos.</li><li>Ahora es posible una frecuencia de exportación en 15 minutos, pero no está disponible de forma predeterminada. Para que esta opción esté disponible en su entorno, primero debe ponerse en contacto con el servicio de atención al cliente de Adobe y solicitar que el grupo de informes esté configurado para admitir exportaciones de 15 minutos.</li></ul><p>**Nota:** con estas mejoras, las URL de las páginas de fuentes de datos en Adobe Analytics también se están actualizando. Actualice los marcadores existentes para que apunten a las nuevas páginas de fuentes de datos antes del 30 de abril de 2026.</p><p>Para obtener más información, consulte [Crear una fuente de datos](/help/export/analytics-data-feed/create-feed.md) y [Administrar fuentes de datos](/help/export/analytics-data-feed/df-manage-feeds.md).</p> | miércoles, 20 de enero de 2026 | miércoles, 24 de febrero de 2026<p>(Originalmente planificado para su lanzamiento el miércoles, 10 de febrero de 2026)</p> |
| **Nueva guía de API para grupos de informes de Adobe Analytics 2.0** <p>Cree, recupere, actualice y elimine las API del grupo de informes. Para obtener más información, consulte la [Guía de referencia de la API del grupo de informes de Adobe Analytics 2.0](https://developer.adobe.com/analytics-apis/docs/2.0/apis/report-suites/) y la [Guía de extremo de la API del grupo de informes de Adobe Analytics 2.0](https://developer.adobe.com/analytics-apis/docs/2.0/guides/endpoints/report-suites/).</p> | | Febrero de 2026 |
| **Nueva guía de informe de detección de anomalías de API de Adobe Analytics 2.0** <p>Cree un informe de detección de anomalías de API automatizado. Para obtener más información, consulte la [Guía de extremo de informe de API de detección de anomalías](https://developer.adobe.com/analytics-apis/docs/2.0/guides/endpoints/reports/anomaly/).</p> | | Febrero de 2026 |
| **Nueva guía de informe de tendencias de fecha básica de API de Adobe Analytics 2.0** <p>Crear un informe de KPI de tendencias de fecha básica de API automatizado Para obtener más información, consulte [Guía de extremo de informe de API de detección de anomalías](https://developer.adobe.com/analytics-apis/docs/2.0/guides/endpoints/reports/kpi/).</p> | | Febrero de 2026 |
